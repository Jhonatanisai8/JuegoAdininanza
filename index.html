Juego de Adivinar un número.

Reglas del juego.

  • La computadora debe seleccionar aleatoriamente un número entre 1 y 100.
  • El usuario debe seleccionar el nivel de dificultad (fácil, medio, difícil) que determinará el número de oportunidades que tiene para adivinar el número.
  • El usuario debe poder introducir su conjetura.
  • Si la suposición del usuario es correcta, el juego debe mostrar un mensaje de felicitación junto con el número de intentos que tomó para adivinar el número.
  • Si la suposición del usuario es incorrecta, el juego debe mostrar un mensaje que indique si el número es mayor o menor que la suposición del usuario.
  • El juego debe terminar cuando el usuario adivina el número correcto o se queda sin oportunidades.
